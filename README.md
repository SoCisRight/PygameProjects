# 1.1 项目介绍
本组成员：吴泽曦（程序）、郭梓涵（美术素材绘制+统一风格）、于千涵（美术素材绘制）、杨雅婷（美术素材绘制）、蓝允余（美术素材绘制）
项目介绍：这是一个横版跑酷游戏，玩家控制小兔子、避开障碍物，吃到胡萝卜即可得分。
玩法介绍：鼠标按键/空格开始游戏。游戏中按下空格键控制小兔跳跃，游戏中草地上有木桩、石块、灌木丛等障碍，除此之外，小兔还要注意避开低空飞行的小鸟。每吃到一根胡萝卜记一分。
